# EES — დღეების კონტროლი

ქართული PWA პიროვნებების ქვეყანაში ყოფნის 90/180-დღიანი წესის სამართავად.

## შესაძლებლობები

- პიროვნების სახელისა და ჩამოსვლის თარიღის დამატება
- მაქსიმალური გასვლის თარიღის გამოთვლა დღეებით
- გასვლის დაფიქსირება და დაბრუნების პირველი შესაძლო თარიღი
- თითოეული პიროვნების შესვლა/გასვლის ისტორია
- ძებნა და მოახლოებული გასვლების შეჯამება
- IndexedDB-ში ლოკალური შენახვა
- სრული JSON Backup და Restore
- Offline მუშაობა და Android/desktop-ზე PWA-დ დაყენება

## ლოკალურად გაშვება

Node.js-ის არსებობისას:

```powershell
node dev-server.cjs
```

შემდეგ გახსენით `http://127.0.0.1:4173`.

## მონაცემები

მონაცემები მხოლოდ მიმდინარე ბრაუზერის IndexedDB-ში ინახება. ბრაუზერის მონაცემების გასუფთავებამდე შექმენით `Backup JSON`. სხვა მოწყობილობაზე გადასატანად გამოიყენეთ `Restore JSON`.
